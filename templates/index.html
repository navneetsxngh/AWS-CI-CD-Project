How it works
A Complete ML Pipeline
📥
Data Ingestion
Automated ingestion splits raw CSVs into train and test sets with full logging and exception tracking.
⚙️
Transformation
Numerical features are scaled; categorical variables are one-hot encoded via a saved preprocessing pipeline.
🏋️
Model Training
Multiple regressors are benchmarked; the best R² model is serialised to artifacts/ automatically.
🔮
Prediction API
A Flask endpoint loads the pickled model and preprocessor to return a real-time predicted math score.
☁️
AWS Beanstalk
Deployed via Elastic Beanstalk with .ebextensions config, zero-downtime updates, and auto-scaling.
📊
EDA Notebooks
Jupyter notebooks provide full exploratory analysis — distributions, correlations, and feature importance charts.
